# Virtual Scope - FPS Mode

A sophisticated overlay scope tool for FPS games with advanced image filtering, motion detection, and anti-detection features.

## ✨ Features

- **Real-time Overlay Scope** - Transparent crosshair overlay for aiming assistance
- **Multiple Visual Filters**:
  - Normal - Standard image view
  - Invert - Night vision style effect
  - Canny Light - Edge detection for enhanced visibility
  - Motion Detection - Highlights moving targets in red
- **Zoom Functionality** - 1x to 6x digital zoom
- **Game Window Detection** - Auto-hide when game loses focus
- **Customizable Hotkeys** - Remap all controls
- **Performance Optimized** - 60 FPS target with dynamic polling
- **Anti-Detection Features** - Memory pattern randomization

## 🎮 Controls

| Action | Default Key |
|--------|-------------|
| Toggle Scope | V |
| Normal Filter | F1 |
| Invert Filter | F2 |
| Canny Light | F3 |
| Motion Detection | F4 |
| Zoom In | + |
| Zoom Out | - |

*All keys can be customized in the UI*

- DirectX compatible graphics
